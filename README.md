# DraperToHerb
Turns all instances of "Kevin Draper" into "Kevin Herb"

So Kevin Draper is a Warriors fan and, as we know, all Warriors fans are [Herbs](http://deadspin.com/warriors-fans-are-herbs-1780139242).
